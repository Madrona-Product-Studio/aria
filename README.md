# Aria Health

An AI health companion that guides people through menopause — not with generic information, but with structured, empathetic, stage-aware support.

Part of [Madrona Product Studio](https://www.madronaproduct.com).

## What it is

A conversational AI constrained to evidence-based health content with real clinical guardrails, stage-aware context, and an empathetic voice. Journey mapping tracks where you are in the menopause transition and delivers phase-specific guidance — not everything at once.

## Stack

- Next.js 15 (App Router), TypeScript, Tailwind CSS
- Claude API with structured system prompt encoding clinical guardrails
- Deployed on Vercel

## Status

Functional prototype — live at [aria-health-mp.vercel.app](https://aria-health-mp.vercel.app). Architecture designed to generalize beyond menopause to any health inflection point.
